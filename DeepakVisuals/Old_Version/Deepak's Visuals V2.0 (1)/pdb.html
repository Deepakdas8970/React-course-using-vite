Discription Of Project

Banking System

This project is a simple banking system web application that allows users to view all customers, transfer money securely between customers, and view the history of all money transfers.

Features

  • View All Customers: Users can see a list of all the customers.
  • Transfer Money: Customers can transfer money securely between each other.
  • Transfer History: Users can view the history of all money transfers.

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • PHP
  • MySQL
  • Bootstrap icon library for icons.

Usage

To use the application, simply open the provided HTML files in a web browser.
